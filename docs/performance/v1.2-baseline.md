# V1.2 Performance Baseline

## V1.1 Production Build Baseline

Measured from commit `34aab55` with Node.js 24 and the production Vite build:

- Initial entry JavaScript: 999,991 bytes (`index-C90MvmDp.js`)
- Initial entry JavaScript gzip output: approximately 279,280 bytes
- Global CSS: approximately 46,730 bytes
- Total `dist/assets`: approximately 4.3 MB, including hidden source maps

The approved V1.2 entry budget is 699,993 bytes, exactly 30% below the V1.1 initial-entry baseline after rounding down to a whole byte.

## Measurement Commands

```bash
npm run build
npm run check:bundle
```

The bundle check reads the initial module script referenced by `dist/index.html`. It deliberately excludes source maps and route chunks because they are not part of the initial page request.
