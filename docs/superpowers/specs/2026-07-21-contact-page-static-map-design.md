# 联系页静态位置地图设计

## 目标

在现有联系信息与留言表单下方增加通栏地图，清晰标注“天水市秦州区安居小区 E 区”，同时保留高德地图外部导航入口。

## 页面布局

- 保持现有左侧联系信息、右侧留言表单的双栏结构不变。
- 在双栏结构下方增加独立的白色圆角地图卡片。
- 卡片顶部展示“办公位置”、完整地址和“在高德地图中打开”链接。
- 地图桌面端高度约 360px，移动端约 280px。

## 技术方案

- 使用独立 `OfficeMap` React 组件展示本地静态 WebP 地图。
- 通过可重复执行的生成脚本下载 OpenStreetMap 瓦片、拼接裁剪并添加公司位置标记。
- 根据高德分享链接的 GCJ-02 中心点转换得到 WGS84 坐标，避免跨地图坐标系造成明显偏移。
- 页面只加载 `/images/office-map.webp`，不在运行时请求第三方地图脚本或瓦片。
- 不新增第三方依赖，生成脚本复用项目已有的 Sharp。

## 稳定性与许可

- 网站运行不需要 API Key、安全密钥或域名白名单。
- 地图图片包含 OpenStreetMap 数据署名。
- 地图顶部保留高德 URI 跳转，供用户查看实时地图和发起导航。

## 验证

- 自动测试覆盖组件接入、本地静态图片、目标地址与高德外部链接。
- 运行 TypeScript 检查、项目测试、生产构建。
- 在本地浏览器验证桌面与移动布局、地图标记及降级显示。
