# Site V1.3 Complete Technical SEO Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make every public route crawlable from raw HTML and provide China-first technical SEO signals compatible with Baidu, Google, and Bing.

**Architecture:** Keep the existing React + Vite client and add a build-only Vite SSR bundle. A shared SEO route manifest drives static HTML rendering, canonical URLs, structured data, sitemap generation, and validation so those outputs cannot drift.

**Tech Stack:** React 18, React Router 7, Vite 6 SSR build, TypeScript, Cheerio, Node test runner, Markdown front matter.

## Global Constraints

- Preserve `BrowserRouter`, route-level lazy loading, the current client bundle budget, and existing public URLs.
- Use `https://lekeopen.com` and trailing slashes for page canonical URLs.
- Include only Markdown records with `publishStatus: published` in generated public SEO routes.
- Set the document language to `zh-CN`; no hreflang matrix is required.
- Add no runtime server, database change, framework migration, or search verification token.
- A production release is complete only after raw HTML, sitemap, robots, RSS, and 404 behavior are verified.
